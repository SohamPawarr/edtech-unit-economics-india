What This Project Is

This is an end-to-end consulting engagement simulation on a fictional Series B EdTech startup — EduVista — operating in the IIT-JEE coaching segment in Tier 2 India. The company is burning ₹3.2 Cr/month with 6 months of runway and a LTV/CAC ratio of 0.8x. The project diagnoses root causes, evaluates three strategic pivot options, builds a month-by-month financial model, and delivers a structured recommendation using the SCR (Situation–Complication–Resolution) framework.
